탐험

// 설치 방법

3단계로 시작하기

별도의 모드 설치 없이 런처 하나로 모든 게 자동으로 처리됩니다.

01
런처 다운로드
아래 다운로드 버튼을 눌러 설치 파일을 받으세요. Windows 전용입니다.
서버 접속기 Setup 1.0.0.exe
02
런처 실행 및 로그인
설치 후 런처를 실행하고 마이크로소프트 계정으로 로그인하세요. Java가 없으면 자동으로 설치됩니다.
03
서버 접속
"서버 접속하기" 버튼을 누르면 모드가 자동으로 다운로드되고 마인크래프트가 실행됩니다. 첫 실행은 10~20분 소요될 수 있습니다.
